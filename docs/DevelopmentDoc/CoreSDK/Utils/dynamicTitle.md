# dynamicTitle.js

`dynamicTitle.js` 用于基于 `sysGlobal` store 的配置动态设置浏览器标题。

## 方法说明

| 方法（带参数）      | 参数说明 | 返回值 | 作用             |
| ------------------- | -------- | ------ | ---------------- |
| `useDynamicTitle()` | 无参数   | `void` | 动态修改页面标题 |

## 说明

1. 依赖 `useSysGlobalStore`。
2. 只适用于当前基座的系统标题配置场景。
