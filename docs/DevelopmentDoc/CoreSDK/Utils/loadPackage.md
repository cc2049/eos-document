# loadPackage.js

`loadPackage.js` 用于在路由切换时动态装载 `form`、`etable`、`vtable`、`workbench` 等子包，是基座初始化链路的一部分。

## 方法说明

| 方法（带参数）                    | 参数说明                                                     | 返回值 | 作用                                   |
| --------------------------------- | ------------------------------------------------------------ | ------ | -------------------------------------- |
| `LoadPackage(router, app, pinia)` | `router`: 路由实例；`app`: Vue 应用实例；`pinia`: pinia 实例 | `void` | 在路由钩子中动态加载子包并完成插件安装 |

## 说明

1. 依赖基座路由、内部 API 和工作台能力。
2. 业务项目不要直接调用，由基座统一接管。
