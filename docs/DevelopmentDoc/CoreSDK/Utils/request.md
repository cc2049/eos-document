# request.js

`request.js` 是基座统一请求入口，内部封装了 token 注入、请求协议包装、加密、下载、错误提示等基础设施能力。

## 主要能力

| 方法（带参数）                            | 参数说明                                                                | 返回值          | 作用             |
| ----------------------------------------- | ----------------------------------------------------------------------- | --------------- | ---------------- |
| `service(config)`                         | `config`: axios 请求配置                                                | `Promise`       | 发起统一协议请求 |
| `download(url, params, filename, config)` | `url`: 下载地址；`params`: 参数；`filename`: 文件名；`config`: 其他配置 | `Promise<void>` | 下载文件         |

## 说明

1. 深度耦合内部协议、鉴权、加密和通知机制。
2. 不应复制到业务项目中另起一套。
