# 数据面板

::: tip 提示
只有需要绑定数据的组件才会显示数据面板，如下拉组件、单选多选组件等。
:::

## 绑定方式 - 接口

| <nobr>属性名称 </nobr> | <nobr> 属性值 </nobr> | 释义                    | 备注           |
| ---------------------- | --------------------- | ----------------------- | -------------- |
| 接口地址               | `INTERURL`            | 接口地址                | 下拉选择       |
| 绑定名称               | `BINDLABEL`           | 数据显示名称            | 固定`LABEL`    |
| 绑定值                 | `BINDVALUE`           | 数据值                  | 固定`VALUE`    |
| 树结构数据             | `ISTREE`              | 是否树结构数据          | 1：是 0：否    |
| 子节点                 | `CHILDREN`            | 树结构数据子节点 key 值 | 固定`CHILDREN` |

#### 接口 - 查询映射方式（数组）

| <nobr>属性名称 </nobr> | <nobr> 属性值 </nobr> | 释义         | 备注                                                              |
| ---------------------- | --------------------- | ------------ | ----------------------------------------------------------------- |
| 中文名称               | `LABEL`               | 字段中文名称 |                                                                   |
| 字段                   | `KEY`                 | 字段         |                                                                   |
| 值类型                 | `VALUETYPE`           | 值的类型     | 1：字段 2：固定值                                                 |
| 值                     | `VALUE`               | 具体值       | 类型为字段时，下拉选择字段。<br/>类型为固定值时，输入框输入字段。 |

## 结果映射关系（数组）

| <nobr>属性名称 </nobr> | <nobr> 属性值 </nobr> | 释义           | 备注       |
| ---------------------- | --------------------- | -------------- | ---------- |
| 被映射字段             | `FIELD`               | 被映射字段     | 下拉选择   |
| 映射字段               | `MAPFIELD`            | 映射字段       | 下拉选择   |
| 只读                   | `READONLY`            | 只读，不可删除 | true/false |
