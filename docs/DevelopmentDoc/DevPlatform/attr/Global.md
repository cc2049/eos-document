# 全局面板

## 全局属性

| <nobr>属性名称 </nobr> | <nobr> 属性值 </nobr> | 释义         | 备注         |
| ---------------------- | --------------------- | ------------ | ------------ |
| 插槽地址               | `INTERURL`            | 接口地址     | 获取数据接口 |
| 页面转菜单             | `CONVERMENU`          | 是否转菜单   | 1：是 0：否  |
| 携带参数               | `ISRELATEID`          | 是否携带参数 | 1：是 0：否  |

## 全局样式

| <nobr>属性名称 </nobr> | <nobr> 属性值 </nobr> | 释义           | 备注               |
| ---------------------- | --------------------- | -------------- | ------------------ |
| 标题对齐               | `LabelPosition`       | 对齐方式       | 顶部/左对齐/右对齐 |
| 值最大宽度             | `ValueWidth`          | 全局值最大宽度 |                    |
