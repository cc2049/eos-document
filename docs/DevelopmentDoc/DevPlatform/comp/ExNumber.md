# 数字

## 后戳类型 `APPEND`

| <nobr>类型 </nobr> | <nobr> 表现 </nobr> | 释义                               |
| ------------------ | ------------------- | ---------------------------------- |
| 无                 |                     | 不显示后戳                         |
| 文字               | 显示输入文字        | 固定显示，无其他效果               |
| 解密               | 显示眼睛图标        | 点击后解密当前组件的加密内容       |
| 超链接             | 显示展示文字        | 点击后跳转至相关配置（路由或外链） |
| 获取验证码         | 显示获取验证码按钮  | 点击获取验证码                     |

## 控制按钮 `CONTROLS`

开启后展示 `+` `-` 控制按钮

## 最小值`MIN`

设置允许的最小值

## 最大值`MAX`

设置允许的最大值

## 数值精度`PRECISION`

小数位数，用于输入与展示

## 步长 `STEP`

设置后，只能输入步长的倍数。也是展示控制按钮时，单击按钮增减的数值。
