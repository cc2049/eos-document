# 子表

## 绑定字段

与后台数据对接使用，不可编辑

## 接口配置

当前表格数据接口配置

## 查询实体

当前表格列信息接口，用于获取配置

## 表格按钮

多选，默认：`新增`，`删除`。不选则不出现表格按钮。

## 表格属性

::: tip 提示
配置查询实体后， 单击组件内的 `+` 按钮，会自动弹出表格配置弹窗，勾选需要显示的列及相关属性配置，即可自动生成表格列。

属性可在弹窗内批量编辑，也可单击列，在右侧面板单独编辑。
:::

| <nobr>属性名称</nobr>         | <nobr> 属性值</nobr> | 释义                 | 备注                                 |
| ----------------------------- | -------------------- | -------------------- | ------------------------------------ |
| 主键                          | `FIELD`              | 控件主键             |                                      |
| 显示名称                      | `LABEL`              | 显示名称             |                                      |
| 显示                          | `ISSHOW`             | 是否显示             |                                      |
| 控件                          | `VTYPE`              | 控件类型             |                                      |
| 超链接                        | `LABELLINK`          | 值是否为超链接       |                                      |
| 支持查询                      | `ISQUERY`            | 是否为查询项         |                                      |
| 支持编辑                      | `ISEDIT`             | 是否可编辑           |                                      |
| 必填                          | `ISREQUIRE`          | 是否为必填项         |                                      |
| 对齐                          | `ALIGN`              | 对齐方式             |                                      |
| 固定方式                      | `FIXED`              | 冻结方式             | `无` , `左` , `右`                   |
| 列宽                          | `WIDTH`              | 当前列宽             | 默认 `120`                           |
| 条件必填                      | `REQUIREEXPRESSION`  | 根据条件判断是否必填 | 例`Data.xxx == 1`                    |
| 条件显示                      | `SHOWEXPRESSION`     | 根据条件判断是否显示 | 例`Data.xxx == 1`                    |
| 计算公式                      | `CALC`               | 显示计算的值         | 例`Data.a + Data.b`                  |
| 合计                          | `SUM`                | 显示合计             | `无` , `求和` , `平均`               |
| 小数点控制                    | `PRECISION`          | 小数位数             |                                      |
| 条件禁用                      | `DISABLEDEXPRESSION` | 根据条件判断是否禁用 | 例`Data.xxx == 1`                    |
| 最小值                        | `MIN`                | 输入的最小值         |                                      |
| 最大值                        | `MAX`                | 输入的最大值         |                                      |
| 菜单 ID                       | `MODULEID`           | 弹窗内展示的菜单 ID  |                                      |
| 页面 ID                       | `PAGEID`             | 弹窗内展示的页面 ID  |                                      |
| 样式                          | `STYLE`              | 弹窗大小             | 例`70%*60%`                          |
| 自定义列字段                  | `COL`                | 自定义列的字段       | 例`BILLNO@VNAME`                     |
| 子表批量回填                  | `MULTIMARGE`         | 子表数据回填         | 开启后数据回填多条，否则回填至一条内 |
| 下拉输入值标识                | `CREATED`            | 下拉组件可添加选项   | 下拉框输入值自动添加为选项           |
| 节点可选择                    | `CHECKSTRICTLY`      | 级联节点任意选择     | 级联组件可以选择任意节点             |
| <nobr>子表启用弹窗表单</nobr> | `ROWFORM`            | 数据弹窗展示         | 行数据启用弹窗展示                   |
| 加密                          | `ISAES`              | 是否加密             |                                      |
| 导出                          | `ISEXPORT`           | 是否可导出           |                                      |
| 唯一                          | `ISONLY`             | 是否唯一             |                                      |
| 合并                          | `ISMERGE`            | 是否合并             |                                      |
