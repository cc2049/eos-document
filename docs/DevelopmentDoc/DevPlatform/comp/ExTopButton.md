# 顶部按钮

::: tip 提示
单击具体按钮，显示当前按钮属性
:::

## 按钮属性

| <nobr>属性名称</nobr> | <nobr> 属性值</nobr> | 释义                   | 备注                     |
| --------------------- | -------------------- | ---------------------- | ------------------------ |
| 按钮类型              | `VTYPE`              | 当前按钮类型           | 下拉选择，如 `保存`      |
| 按钮名称              | `VNAME`              | 当前按钮名称           | 可手动修改               |
| 接口绑定              | `ACTIONADDRESS`      | 当前按钮绑定接口       | 下拉选择                 |
| 事件类型              | `BtnType`            | 当前按钮事件类型       | 下拉选择，如 `提交数据`  |
| 按钮排序              | `SORTCODE`           | 当前按钮在按钮组内排序 | 手动输入                 |
| 禁用条件              | `DisableCond`        | 当前按钮禁用条件       | 手动输入                 |
| 禁用描述              | `DisableDesc`        | 当前按钮禁用描述       | 手动输入，点击后横幅展示 |
