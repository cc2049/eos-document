# App 基座 - 版本升级文档

## 4.1.260301
### 升级内容概览
#### 新增功能
- 引入全新的深色模式主题，支持用户一键切换，提升夜间阅读体验。
- 优化了表单验证逻辑，新增实时异步校验接口，减少用户提交错误率。
- 在个人中心页面增加了“操作日志”模块，用户可查看最近30天的关键操作记录。
- 适配了最新的 iOS 和 Android 系统版本，解决了部分移动端兼容性问题。

#### 修复功能
- 修复了在弱网环境下，图片加载失败后未显示默认占位图的问题。
- 解决了多标签页同时操作时，本地存储（LocalStorage）数据不同步导致的异常报错。
- 修正了日期选择器在跨年份选择时，月份计算偏移一天的 Bug。
- 修复了特定分辨率下，导航栏菜单遮挡页面主要内容区域的样式问题。
